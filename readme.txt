How to build up your own website?

1. Create a public github repository named as 'bqwang97.github.io' (mandatory)

2. Upload this local repository to the github

3. Open repo bqwang97.github.io's settings, go to 'Pages'

4. In 'Branch' section, set 'main/root' as your main branch, then click 'Save'

5. After saving the branch, wait for about 30s and refresh it, your website url should be shown as 'https://bqwang97.github.io' in the settings.

6. Happy Birthday!
